# 💒 Nos Casamos — Jovita & Joaquin

Página web elegante e interactiva para la invitación a la boda de Jovita y Joaquin.

## 📋 Descripción

Sitio web responsivo diseñado especialmente para invitar a los seres queridos a la boda. Cuenta con un diseño minimalista y elegante, con animaciones suaves y una experiencia de usuario optimizada para todos los dispositivos.

**Fecha de la boda:** 14 de noviembre de 2026  
**Hora:** 14:00

## ✨ Características

- **Navbar responsivo** con menú hamburguesa para dispositivos móviles
- **Barra de progreso visual** que indica el scroll de la página
- **Contador regresivo** animado hacia la fecha de la boda
- **Secciones temáticas:**
  - Portada con imagen de fondo y presentación elegante
  - Contador de días, horas, minutos y segundos
  - Itinerario del evento
  - Ubicación con información de localización
  - Formulario de RSVP
  - Galería de fotos
- **Diseño responsivo** optimizado para desktop, tablet y móvil
- **Animaciones suaves** con transiciones CSS fluidas
- **Tipografía elegante** con fuentes Google Fonts (Cormorant Garamond y Jost)
- **Paleta de colores sofisticada:**
  - Azul cielo profundo
  - Verde pistache
  - Dorado elegante
  - Grises neutros

## 🚀 Cómo Usar

1. **Descargar o clonar** el proyecto en tu máquina
2. **Abrir** `index.html` en tu navegador web favorito
3. **Personalizar** con tus imágenes y detalles específicos

### Modificaciones Recomendadas

- Cambiar las imágenes en la carpeta `assets/images/`
- Actualizar los horarios y detalles del itinerario en el HTML
- Modificar el correo de contacto en la sección RSVP
- Agregar fotos del evento en la galería

## 📄 Funcionalidades JavaScript

### `inicializarNavbar()`
Configura el menú hamburguesa responsivo y el efecto scroll del navbar

### `inicializarScrollProgress()`
Implementa la barra de progreso visual en la parte superior

### `iniciarContador()`
Calcula y actualiza el contador regresivo cada segundo

### `agregarEfectosHover()`
Añade transiciones suaves a las tarjetas interactivas

### `agregarAnimacionesCSSEntrada()`
Implementa animaciones de entrada con Intersection Observer
